# 数组
## 声明
数组内部由一组有序元素组成。每一个元素可以保存一个值。可以通过一个正整数索引值引用数组中的元素。

在OX中，可以用以下语句声明一个数组：
```
[0,1,2,3]
```
上面的语句声明了一个4元素的数组，在"["和"]"中为数组的元素值表达式，每个元素值表达式通过","分隔。
如果一个元素表达式为空，则表示这个元素值为null。比如：
```
[0,,1] //等价于[0,null,1]
```
如果通过多行声明数组，元素在不同行声明，可以省略","。比如以下语句和上面的数组声明语句是完全等价的：
```
[
    0
    1
    2
    3
]
```
## 元素展开
我们可以通过元素展开将一个数组的内容添加到另一个数组中。看下面的代码：
```
a = [1,2,3] //定义变量a，其值为数组[1,2,3]
b = [0, ...a] //定义变量b，将数组a的元素体添加到b中，b的值为[0,1,2,3]
```
## 元素追加
如果我们已经定义了一个数组变量，可以通过元素追加语句向数组中添加新元素。
```
a = [] //声明变量a为一个0元素数组
a.[0,1,2] //向a中追加元素，a的值变为[0,1,2]
b = [5,6]
a.[3,4,...b] //向a中追加元素，a的值变为[0,1,2,3,4,5,6]
```

语法描述：
```
array_literal: "[" array_items? "]"

array_items: array_items "," array_item
    | array_item

array_item:
    | expression
    | "..." expression

array_append_items: expression "." "[" array_items? "]"
```
## 数组长度
我们可以通过访问数组的"length"属性，获取数组中元素的个数。
```
a = []
stdout.puts("{a.length}\n") //打印0
a.[1,2,3]
stdout.puts("{a.length"\n") //打印3
```
设置"length"属性会修改数组的长度，如果新长度小于原长度，数组被截短，如果新长度大于原长度，数组会扩张，新增元素被初始化为null。
