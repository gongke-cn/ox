# **OX语言**
## 简介
OX语言([https://gitee.com/gongke1978/ox](https://gitee.com/gongke1978/ox))是一个新型脚本语言。
可以用OX语言简化复杂的开发过程，或将OX语言作为内置脚本引擎集成到其他程序中。
## 特性
* 面向对象
* 通过垃圾收集器进行内存管理
* 内置正则表达式支持
* 内置迭代器支持
* 文字模板功能
* 线程和纤程支持
* C语言调用接口
* 内置模块管理功能
* 内置多语言支持
* 无须编译直接解释执行
