# **OX Language**
## Introduction
OX Language([https://gitee.com/gongke1978/ox](https://gitee.com/gongke1978/ox)) is a new scripting language.
You can use OX Language to simplify complex development processes, or integrate it as a built-in script engine into other programs.
## Features
* Object-Oriented
* Memory Management Through Garbage Collector
* Built-in Regular Expression Support
* Built-in Iterator Support
* Text Template Functionality
* Thread and Fiber Support
* C Language Calling Interface
* Built-in Module Management Functionality
* Built-in Multi-language Support
* Direct Interpretation and Execution Without Compilation
